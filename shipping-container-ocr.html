Shipping Container OCR

Adapt AI’s Shipping Container OCR is a cutting-edge, AI-powered solution designed to accurately decode shipping container codes in real-time. Built for developers and businesses, this system excels in challenging environments—handling blurry, low-resolution, dark, over-exposed, angled, or rusty containers with unparalleled precision.

Key Features

High Accuracy

Recognizes container codes in diverse conditions, including blurry images, low resolution, dark settings, over-exposure, tough angles, shadow marks, rusty text, and tank containers.

Fast Deployment

Seamlessly process live camera feeds (RTSP), video files, or static images. Deploy on-premise via Docker on x86 devices (no GPU required) or use our cloud service with JSON, webhooks, or CSV outputs.

Developer-Friendly

Includes an SDK for on-premise image processing (3 MB/image limit, 8 calls/sec in cloud) and fast inference speeds, making integration effortless for developers.

Detailed Results

Outputs include Owner & Product Group Code (4 letters), Registration Number (6 digits), Check Number (1 digit), and a confidence score for each recognition.

Applications

  • Logistics & Shipping: Automate container tracking and inventory management at ports and warehouses.

  • Security: Enhance gate automation and access control with real-time container identification.

  • Supply Chain Optimization: Streamline operations by integrating with existing systems via API or webhooks.

Technical Specifications

- Live Camera: Processes RTSP feeds for real-time monitoring.
- Video Processing: Analyzes video files programmatically in bulk.
- Image Processing: Supports up to 3 MB images with rapid inference.
- Deployment Options: On-premise (Docker, x86) or cloud-hosted.
- Output Formats: JSON, CSV, or webhooks for easy integration.
